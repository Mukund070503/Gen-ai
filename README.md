# 🤖 RAG Chatbot (LangChain + Pinecone + Gemini)

This project implements a Retrieval-Augmented Generation (RAG) chatbot using:

- LangChain
- Pinecone (Vector DB)
- Google Gemini API
- PDF document ingestion

## 🚀 Features

- Upload PDF and create embeddings
- Store vectors in Pinecone
- Ask questions based on document
- Context-aware answers

## 🧠 Tech Stack

- Node.js
- LangChain
- Pinecone
- Gemini API
